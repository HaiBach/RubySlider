QUICK START

Quick start will give the basics to run codetabs. The other sections will provide more detailed options.

1. Upload code folder to your host, this folder included core script, style, and image of codetabs to work

  • code

Other folder for reference

  • documentation | this user manual
  • example | basic example for codetabs
  • template | use codetabs in theme, more design and function
  • libs | js, css, image... → using in doc, example and template


2. CODETABS FILE : add main js, css of codetabs into page :

<!-- Main style codeslider, included default skin -->
<link rel="stylesheet" href="code/codetabs.css">

<!-- Effect animation core in tabs -->
<link rel="stylesheet" href="code/code.animate.css">

<!-- jQuery 1.7+ required - hosted from Google -->
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

<!-- Main js codeslider -->
<script src="code/codetabs.js"></script>


3. Add slides to codetabs.

  • Width-slider always equal to width-container
  • .ct → core style of tabs.     .ct-flatbox → tabs using style flatbox.
<div class="ct ct-flatbox">
    <!-- TAB 1 -->
    <div>
        <!-- Tab title -->
        <div class="cs-pagitem">Tabs title</div>

        <!-- Tab content -->
        <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit...</p>
    </div>

    <!-- TAB others same TAB 1 -->
    <div>...</div>
</div>


4. Initialize the codetabs

<script>
    jQuery(document).ready(function($) {

        $('.ct').codetabs({
            speed   : 800,
            fxInOut : 'glueHor',
            pag     : { pull : 'center' }
        });
    });
</script>
With few simple steps, you have created codetabs working!

Notes

  • Phan biet value type bang mau sac:
    number - string - boolean - mixed value

  • HTML5 data - De dang thiet lap tuy chon bang Object literal notation giong nhu cac plugin jQuery khac, dong thoi ho tro thiet lap tuy chon thong qua HTML5 data attributes.

    Cau truc tuy chon trong HTML5 data : [key]-[value], cac tuy chon duoc phan cach boi khoang trang.

    HTML5 DATA 
    <!-- Setup options direct on tabs -->
<div class="ct" data-codetabs="fxOne-glueHor speed-1200 pag-{ pull: center }">
    ...
</div>
    JAVASCRIPT 
    // Setup options in javascript
$('.ct').codetabs({
    fxOne   : 'glueHor',
    speed   : 1200,
    pag     : { pull : 'center' } 
});

  • Tuy chon tren slide - duoc thiet lap truc tiep tren moi slide thong qua data-slide (HTML5 data attributes)
    Codetabs se uu tien lay tuy chon tren slide voi cung ten goi so voi tuy chon chung tren codetabs.

    <div class="cs" data-codetabs="speed-400 delay-5000">
    
    <!-- slide custom setup -->
    <div data-slide="speed-600 delay-8000"> ... </div>
    ...
</div>
  • Options have is prefix → boolean value.
    true/on/1 same result for true boolean value in HTML5 data attributes.
    false/off/0 same result for false boolean value in HTML5 data attributes.

POSITIONS

Voi su ket hop 3 options: 'dirs', 'pos', 'pull' -> tao ra 14 vi tri khac nhau cho tabs

  • dirs : 'hor', 'ver'
  • pos  : 'top', 'bottom'
  • pull : 'begin', 'center', 'end', 'justified'

DIRECTIONS

Co 2 huong trong Codetabs: 'horizontal' va 'vertical'

Huong horizontal la huong mac dinh trong codetabs, ban khoi can setup.
Chuyen sang tabs vertical, thay tu khoa 'hor' sang 'ver'

HTML5 DATA 
<!-- Direction HORIZONTAL setup -->
<div class="ct" data-codetabs="pag-{ dirs: hor }"> ... </div>

<!-- Direction VERTICAL setup -->
<div class="ct" data-codetabs="pag-{ dirs: ver }"> ... </div>
JAVASCRIPT 
// Direction HORIZONTAL setup
$('.ct').codeslider({ pag: { dirs: 'hor' } });

// Direction VERTICAL setup
$('.ct').codeslider({ pag: { dirs: 'ver' } });

SLIDESHOW

BASIC SETUP

  • Bat sildeshow voi option 'isSlideshow : true', mac dinh la tinh nang slideshow tat di.
  • Setup thoi gian chuyen doi giua cac slide voi option 'delay' options Doi tuong 'pag', mac dinh la 8000 milliseconds(ms)
  • Them style cho timer bang class. i.e. '.ct-arcline'
HTML5 DATA 
<!-- Combine add style for timer and turn on slideshow -->
<div class="ct ct-arcline" data-codetabs="isSlideshow-true slideshow-{ delay: 5000 }"> ... </div>
JAVASCRIPT 
<!-- Setup style for timer -->
<div class="ct ct-arcline"> ... </div>
// Setup option for slideshow
$('.ct').codetabs({
    isSlideshow : true,
    slideshow   : { delay : 5000 }
});

<!-- Setup style for timer -->
<div class="ct ct-arcline"> ... </div>
// Setup option for timer
$('.ct').codetabs({
    slideshow : { timer : 'arc' }
});

Timer arc customize

  • Style sheet '.ct-arcline' : tuy bien vi tri kich thuoc cua timer, ket hop voi button play/pause
  • Options code: tuy bien mau sac, do day, kich thuoc vong tron trong timer
Timer arc options

Arc bao gom 2 vong tron inner va outer, ca 2 deu co 4 thuoc tinh giong nhau trong options Doi tuong 'arc'
Vong tron Inner de hien thi thoi gian cua slideshow
Vong tron Outer de lam nen cho Inner hoac trang tri

Variable Default value Description
width null Chieu rong cua timer
height null Chieu cao cua timer
update 30 So lan cap nhap timer trong 1 giay.
rotate 0 Goc bat dau xoay cua duong vien vong trong o trong
radius 7 Ban kinh cua cua vong tron o trong
weight 2 Do day duong vien cua vong tron o trong
stroke #0cf Mau sac duong vien cua vong tron o trong, dinh dang giong trong css
fill transparent Mau sac nen cua vong trong o trong, dinh dang giong trong css
oRadius 7 tuong tu 'radius', nhung cho vong tron o ngoai
oWeight 2 tuong tu 'weigth', nhung cho vong tron o ngoai
oStroke transparent tuong tu 'stroke', nhung cho vong tron o ngoai
oFill transparent tuong tu 'fill', nhung cho vong tron o ngoai

Code sample cho customize timer arc

HTML5 DATA 
<div class="ct ct-arcline" data-codetabs="
    arc-{ stroke: #0cf, radius: 14, oStroke: #e5e5e5, oRadius: 14, oWeight: 2 }
    "> ... </div>
JAVASCRIPT 
$('.ct').codetabs({
    arc : { 
            stroke  : '#0cf',
            radius  : 14,
            oStroke : '#e5e5e5',
            oRadius : 14,
            oWeight : 2
          }
});

code.prev();        // Previous slide
code.goTo(id);      // Goto slide id number, begin 0

code.prop(opts [, isNoRefresh]);    // Auto update options and refresh tabs

code.refresh();                     // Refresh tabs

code.play();        // play slideshow
code.pause();       // pause slideshow
code.stop();        // stop slideshow
