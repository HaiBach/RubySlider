FEATURES
 
3 Layout slider

Support multiply layout, style as much as possible.

 
Responsive

Support fully responsive and non-responsive.

 
CSS3 transition

Priority use css3 transitions instead of jquery animate → reduced memory, faster performance.

 
Touch friendly

Easy goto prev/next slide with intuitive touch/drag.

 
Image lazyload

Show slider after loaded first slide, without waiting loaded all slide.

 
Image auto fit/fill

Image auto fit/fill slide. Support for images of different sizes.

 
Image center

If image size smaller/larger than slide, it's always in center of the slide.

 
Layout mutual support

Switch layout, fallback layout custom when not brower not support css3 transition.

 
Fullwidth helper

With mCanvas & media, setup fullwidth becomes more easier.

 
Show/hide slider

Support show/hide slider depend on width page, or device.

 
Layer support

More vivid with layer, and css3 transform and easy setup.

 
Layer multi transition

Support multi transition with unlimited number.
Repeat and direction animation.

 
Autoplay

Auto play when slide load complete. Auto pause when go out page, or scroll out sider

 
Markup flexible

Can use div or unordered list tag for slide. Position of elements can be in/outside slider...

 
Update options

Updated most of slider options.
